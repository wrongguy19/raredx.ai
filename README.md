# 🧬 RareDx AI v2.0 — Rare Disease Diagnosis System

A **4-Layer AI diagnostic support tool** for rare diseases, built for MBA Term 8 Project.
Uses the Orphanet database + HPO symptom ontology to help clinicians identify candidate rare diseases
based on patient clinical profiles.

---

## 🔍 Features

| Layer | Method | Description |
|-------|--------|-------------|
| L1 | Cosine Similarity | Base retrieval across 7,000+ rare diseases |
| L2 | Disease Clustering | 15 disease clusters for focused search |
| L3 | Knowledge Graph | Gene-based disease relationship links |
| L4 | Anomaly Detection | Ultra-rare / outlier profile flagging |

---

## 🚀 Getting Started

### 1. Install dependencies
```bash
pip install streamlit scikit-learn pandas numpy
```

### 2. Download data files
Place the following raw data files in the project root (not included in repo due to size):
- `rare_diseases_complete.csv`
- `rare_diseases_genes.csv`
- `rare_diseases_info.csv`
- `rare_diseases_natural_history.csv`
- `rare_diseases_prevalence.csv`
- `phenotype.hpoa` — from [HPO Downloads](https://hpo.jax.org/app/data/annotations)

Source: [Orphanet / orphadata.com](https://www.orphadata.com/rare-diseases/)

### 3. Run the pipeline (in order)

```bash
python 1_Data_Prep.py
python 2_Feature_Engineering.py
python step3_model_training.py
python step3b_clustering.py
python step3c_knowledge_graph.py
python step3d_anomaly.py
```

### 4. Launch the doctor interface

```bash
streamlit run step4_doctor_interface.py
```
